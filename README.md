# merchant-sdk-php
This is official PHP SDK for the [merchant-sdk](https://docs.litehex.com/merchant-sdk)

## Installation
```
# composer require litehex/merchant-sdk-php
```

## Requirements
```json
{
   "php": ">=8.1",
   "ext-curl": "*",
   "ext-mbstring": "*"
}
```
